# Human Activity Recognition Using GRU Networks

Human activity recognition system using recurrent neural networks and inertial sensor data.

## Overview

The objective of this project was to classify human activities from accelerometer and gyroscope measurements using recurrent neural networks. Sensor data from multiple subjects were collected, preprocessed, and used to train GRU and Bidirectional GRU models for activity recognition.

## Features

- Collected and processed accelerometer and gyroscope measurements.
- Generated sliding-window sequences for time-series classification.
- Applied feature scaling and label encoding.
- Implemented GRU and Bidirectional GRU models.
- Performed hyperparameter optimization.
- Used k-fold cross-validation for model evaluation.
- Applied early stopping and learning-rate scheduling.
- Evaluated models using confusion matrices.
- Visualized activity predictions and sensor signals.
- Classified activities from multiple subjects.
